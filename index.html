选择网页备份站
OpenSearch 语法
本页支持 OpenSearch 协议，可将"随备"添加为浏览器搜索引擎，在地址栏直接搜索网页备份。
添加搜索引擎
Chrome / Edge / Firefox：打开本页后，在浏览器地址栏右侧点击"+"或进入设置 → 搜索引擎 → 添加，即可将"随备"设为搜索引擎。之后在地址栏输入关键词即可直达搜索结果。
URL 参数
参数说明必填
q要查询或保存的网址是
s目标备份站 ID（见下表）否，默认 wayback
a执行动作（latest / all / save）否，默认 latest
备份站 ID 对照表（s 参数）
ID备份站
waybackWayback Machine（默认）
archiveitArchive-It
archivetodayArchive.today
ghostarchiveGhost Archive
megalodonMegalodon.jp
arquivoArquivo.pt
动作类型（a 参数）
值说明
latest查看最新备份（默认）
all查看所有备份记录
save保存网页到备份站
使用示例
查看最新备份（Wayback Machine，默认）
/?q=example.com · 尝试
查看 Archive.today 上的备份
/?q=example.com&s=archivetoday · 尝试
保存网页到 Wayback Machine
/?q=example.com&a=save · 尝试
保存网页到 Megalodon.jp
/?q=example.com&s=megalodon&a=save · 尝试
查看 Ghost Archive 上所有备份
/?q=example.com&s=ghostarchive&a=all · 尝试
说明
· 通过 URL 参数访问时，页面会在当前标签页直接跳转到目标备份站（避免浏览器弹窗拦截）。
· 使用 Megalodon.jp 时，若网址不含 http:// 或 https://，系统会自动补全为 https://。
· Ghost Archive 与 Arquivo.pt 的保存功能需在其网站内手动确认，因此点击"保存"会跳转至对应首页。
